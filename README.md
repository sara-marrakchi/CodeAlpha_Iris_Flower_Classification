# CodeAlpha_Iris_Flower_Classification
Machine Learning project for Iris Flower Classification using Decision Tree Classifier.

This project uses a **Decision Tree Classifier** to accurately classify Iris flowers based on their sepal and petal measurements.

##  Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

##  Results
- Dataset: 150 samples
- Model: Decision Tree Classifier
- Accuracy: **100%**
